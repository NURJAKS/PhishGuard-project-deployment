# 🔍 Полный анти-фишинговый аудит

## Описание

Система теперь поддерживает **полный анти-фишинговый аудит** с детальным анализом всех аспектов сайта через Gemini AI.

## Что анализируется

### 1. URL-анализ
- Подозрительные символы (кириллица вместо латиницы, похожие символы)
- Подозрительные поддомены
- Длинные или сложные ссылки
- Маскировка настоящего адреса

### 2. Возраст домена
- Новый домен (<90 дней) = высокий риск
- Старый домен = низкий риск
- Используется WHOIS API для определения возраста

### 3. TLS-сертификат
- Кто выдал сертификат (доверенный/недоверенный)
- Срок действия сертификата
- Совпадает ли CN/SAN с доменом
- Проверка валидности сертификата

### 4. DNS и хостинг
- Страна сервера (подозрительные страны?)
- IP-репутация (известный хостинг или подозрительный?)
- Необычные NS или MX записи
- Геолокация сервера

### 5. Структура HTML/JS страницы
- Признаки копирования официального сайта
- Фальш-формы для ввода данных
- Скрытый JavaScript
- Фейковые API вызовы
- Подозрительные скрипты

### 6. Контент-анализ
- Ошибки в тексте (опечатки, грамматика)
- Копированные изображения (логотипы)
- Фальш-логотипы известных брендов
- Несоответствие контента заявленному бренду

### 7. Опасное поведение
- Подозрительные редиректы
- Формы отправки данных на внешние серверы
- Вредоносные скрипты
- Сбор чувствительных данных

### 8. Вывод
- Вердикт: "Безопасно" / "Подозрительно" / "Опасно"
- Короткое объяснение (2-3 предложения)
- Вероятность фишинга в % (0-100)

## Требования

1. **Gemini AI API ключ** (см. `GEMINI_API_SETUP.md`)
2. **Дополнительные библиотеки** (опционально, для DNS):
   ```bash
   pip install dnspython
   ```

## Использование

### Через API:
```bash
curl -X POST http://localhost:8000/v1/ai/full-audit \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```

### В коде:
```python
from ai_analyzer import analyze_url_full_audit

result = analyze_url_full_audit("https://example.com")
print(f"Вердикт: {result['verdict']}")
print(f"Вероятность фишинга: {result['phishing_probability']}%")
```

## Преимущества

✅ **Детальный анализ** - проверяет все аспекты сайта
✅ **Автоматическое сравнение** - сравнивает с оригинальными источниками
✅ **Структурированный отчет** - четкий формат по всем пунктам
✅ **Точная оценка риска** - процент вероятности фишинга
✅ **Использует Gemini AI** - мощный AI для анализа

## Пример использования в расширении

Можно добавить кнопку "Полный аудит" в расширение, которая будет вызывать этот endpoint и показывать детальный отчет пользователю.
