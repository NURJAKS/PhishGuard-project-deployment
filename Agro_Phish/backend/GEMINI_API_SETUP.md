# 🔑 Настройка Gemini AI API ключа

## Как получить свой API ключ Gemini AI

1. **Перейдите на Google AI Studio**: https://aistudio.google.com/
2. **Войдите** в свой Google аккаунт
3. **Создайте API ключ**:
   - Нажмите "Get API key" или "Получить API ключ"
   - Выберите проект или создайте новый
   - Скопируйте созданный API ключ

## Как установить API ключ

### Вариант 1: Через переменную окружения (РЕКОМЕНДУЕТСЯ)

```bash
# Linux/Mac
export GOOGLE_API_KEY="ваш_ключ_здесь"

# Windows (PowerShell)
$env:GOOGLE_API_KEY="ваш_ключ_здесь"

# Windows (CMD)
set GOOGLE_API_KEY=ваш_ключ_здесь
```

### Вариант 2: В файле .env (если используется python-dotenv)

Создайте файл `.env` в папке `backend/`:
```
GOOGLE_API_KEY=ваш_ключ_здесь
```

### Вариант 3: Прямо в коде (НЕ РЕКОМЕНДУЕТСЯ для продакшена)

Отредактируйте `ai_analyzer.py` и замените `DEFAULT_GOOGLE_API_KEY` на ваш ключ.

## Проверка работы

После установки ключа перезапустите backend сервер:

```bash
cd Agro_Phish/backend
python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Проверьте логи - должно появиться:
```
[AI API] API Key source: ENV (masked: AIzaSyDm-C...)
```

## Что изменилось

✅ Система теперь использует **улучшенный промпт** для Gemini AI, который:
- **Сравнивает сайты с оригинальными источниками** (банки, платежные системы)
- **Определяет фальшивые сайты** по несоответствию адресов
- **Дает более точные результаты** благодаря сравнению с базой официальных сайтов

✅ База официальных источников находится в `official_sources.json`:
- Банки (Halyk, Kaspi, Sberbank и др.)
- Платежные системы (Stripe, PayPal)
- IT компании (Google, Microsoft)

Вы можете добавить свои официальные источники в этот файл!

## Пример работы

Когда пользователь проверяет сайт `halykbank-kz.netlify.app`:
1. Система находит, что это похоже на Halyk Bank
2. Сравнивает адрес с официальными: `halykbank.kz`, `www.halykbank.kz`
3. Обнаруживает несоответствие
4. Помечает как **ФАЛЬШИВКА** с высоким риском
