## 🔄 Как перезагрузить расширение:

### Вариант 1: Через интерфейс Chrome
1. Откройте `chrome://extensions/`
2. Найдите расширение **PhishGuard**
3. Нажмите кнопку **🔄 Обновить** (или переключите выключатель OFF → ON)

### Вариант 2: Полная перезагрузка
1. Откройте `chrome://extensions/`
2. Найдите расширение **PhishGuard**
3. Нажмите **Удалить** (Remove)
4. Нажмите **Загрузить распакованное** (Load unpacked)
5. Выберите папку: `/home/nurbol/new_phisss/obnowa_phishiGUARDI/Agro_Phish/extension`

## 🔍 Проверка работы:

1. **Откройте popup расширения** (кликните на иконку)
2. **Откройте DevTools для popup:**
   - Правый клик на popup → "Проверить элемент" (Inspect)
   - Или нажмите F12 когда popup открыт
3. **Проверьте консоль:**
   - Должны быть логи: `PhishGuard popup loaded`
   - Должен быть лог: `UI initialized successfully`
   - Не должно быть ошибок

## 🐛 Если все еще загружается:

1. **Проверьте консоль браузера** (F12) на наличие ошибок
2. **Проверьте что backend работает:**
   ```bash
   curl http://localhost:8002/health
   ```
3. **Проверьте что порт правильный:**
   - В консоли должно быть: `Making direct API call to: http://localhost:8002/...`
