# 🚀 Как запустить PhishGuard Backend

## Проблема: "Не удалось проверить страницу" или "нет соединения с интернетом"

Эта ошибка появляется, когда **backend сервер не запущен**. Расширение пытается подключиться к `http://localhost:8000`, но сервер недоступен.

## Решение

### Шаг 1: Установите pip (если его нет)

```bash
sudo apt update
sudo apt install python3-pip
```

### Шаг 2: Установите зависимости

```bash
cd Agro_Phish/backend
pip3 install -r requirements.txt
```

**Примечание:** Если получаете ошибку прав доступа, используйте `--user`:
```bash
pip3 install -r requirements.txt --user
```

### Шаг 3: Запустите backend сервер

**Вариант А: Используйте скрипт запуска**
```bash
cd Agro_Phish
./start_backend.sh
```

**Вариант Б: Запустите вручную**
```bash
cd Agro_Phish/backend
python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Шаг 4: Проверьте, что сервер запущен

Откройте в браузере: http://localhost:8000/health

Должен появиться ответ `{"status":"ok"}`

### Шаг 5: Перезагрузите расширение

1. Откройте `chrome://extensions`
2. Найдите PhishGuard
3. Нажмите кнопку обновления (🔄)

## Проверка работы

После запуска backend:
- Откройте любую страницу в браузере
- Нажмите на иконку расширения PhishGuard
- Должна появиться проверка страницы без ошибок

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.

## Решение проблем

### Ошибка "ModuleNotFoundError"
- Убедитесь, что установили все зависимости: `pip3 install -r requirements.txt`

### Ошибка "Address already in use"
- Порт 8000 уже занят. Остановите другой процесс или измените порт:
  ```bash
  python3 -m uvicorn main:app --reload --host 0.0.0.0 --port 8001
  ```
  Затем измените конфигурацию расширения на порт 8001.

### Ошибка "Permission denied"
- Используйте `--user` флаг при установке:
  ```bash
  pip3 install -r requirements.txt --user
  ```

## Автоматический запуск при загрузке системы

Создайте systemd service для автоматического запуска:

```bash
sudo nano /etc/systemd/system/phishguard.service
```

Добавьте:
```ini
[Unit]
Description=PhishGuard Backend
After=network.target

[Service]
Type=simple
User=ваш_пользователь
WorkingDirectory=/home/nurbol/Загрузки/PhishGuard-main(1)/PhishGuard-main/Agro_Phish/backend
ExecStart=/usr/bin/python3 -m uvicorn main:app --host 0.0.0.0 --port 8000
Restart=always

[Install]
WantedBy=multi-user.target
```

Затем:
```bash
sudo systemctl enable phishguard
sudo systemctl start phishguard
```
