# Как запустить PhishGuard C#

## Требования

- .NET 8.0 SDK или выше
- Windows, Linux или macOS

## Установка .NET SDK

### Linux (Ubuntu/Debian)
```bash
wget https://dot.net/v1/dotnet-install.sh
chmod +x dotnet-install.sh
./dotnet-install.sh --channel 8.0
export PATH=$PATH:$HOME/.dotnet
```

### Windows
Скачайте и установите с https://dotnet.microsoft.com/download

### macOS
```bash
brew install dotnet
```

## Запуск проекта

1. Перейдите в директорию проекта:
```bash
cd PhishGuardCSharp/PhishGuard.API
```

2. Восстановите зависимости:
```bash
dotnet restore
```

3. Запустите проект:
```bash
dotnet run
```

Или для разработки с hot reload:
```bash
dotnet watch run
```

## Доступ к API

- API: http://localhost:8000
- Swagger UI: http://localhost:8000/swagger
- Health Check: http://localhost:8000/v1/check/health

## Структура проекта

```
PhishGuardCSharp/
├── PhishGuard.API/          # Web API приложение
├── PhishGuard.Core/         # DTOs и общие модели
├── PhishGuard.Data/         # Entity Framework модели
└── PhishGuard.Services/     # Бизнес-логика
```

## База данных

База данных SQLite создается автоматически в папке `PhishGuard.API/database/phishguard.db` при первом запуске.

## Основные API endpoints

### Проверка URL
```bash
curl -X POST http://localhost:8000/v1/check/url \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'
```

### Получить статистику
```bash
curl http://localhost:8000/v1/check/incidents/stats
```

### Управление черным списком
```bash
# Получить список
curl http://localhost:8000/admin/blacklist

# Добавить домен
curl -X POST http://localhost:8000/admin/blacklist \
  -H "Content-Type: application/json" \
  -d '{"domain": "example.com"}'

# Удалить домен
curl -X DELETE http://localhost:8000/admin/blacklist/example.com
```

## Сборка для продакшена

```bash
dotnet publish -c Release -o ./publish
```

Затем запустите:
```bash
./publish/PhishGuard.API
```
