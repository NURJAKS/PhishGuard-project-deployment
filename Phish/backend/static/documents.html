Загрузка документа для анализа

Перетащите документ (или архив ZIP) в эту область или нажмите для выбора файла.
Поддерживаемые форматы: PDF (в т.ч. PDF/A), DOC/DOCX, XLS/XLSX, PPT/PPTX, ODT/ODS/ODP, JPG/PNG/TIFF/HEIC, XML (в т.ч. подписанный XML), CSV, ZIP (пакеты документов).
Ограничения: до 25MB на файл; анализ ZIP выполняется с лимитами для защиты от злоупотреблений архивами.
